# Energy Meter Price Prediction using Multiple Algorithms

**Problem Type:** Regression  
**Objective:** Compare the performance of different machine learning algorithms
for predicting energy meter prices.

## Project Overview
This project focuses on predicting the price of an energy meter based on
relevant input features. Multiple machine learning algorithms are implemented
and compared to evaluate their prediction performance.

## Algorithms Used
- Naive Bayes (NB)
- Linear Discriminant Analysis (LDA)
- Support Vector Machine (SVM)
- Logistic Regression (LR)
- K-Nearest Neighbors (KNN)
- Classification and Regression Tree (CART)

## Approach
- Data loading and preprocessing
- Feature analysis and preparation
- Training multiple machine learning models
- Comparing model performance using evaluation metrics


## Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Results
The project demonstrates how different machine learning algorithms perform on
the same dataset and highlights their strengths and weaknesses in predicting
energy meter prices.

## Notes
This project is intended for learning purposes to understand algorithm
comparison and model evaluation techniques.
